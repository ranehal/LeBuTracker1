# Shwapno Price Tracker

A simple web app to track prices of products from Shwapno. It includes a scraper to get the latest data and a frontend to visualize price history and find deals.

## Features
- Price history tracking with charts
- Search and category filters
- Daily automated updates via script
- Mobile responsive UI

## How to use
Open `index.html` in your browser.
For the scraper, you'll need Python and the dependencies in `requirements.txt`.

```bash
pip install -r requirements.txt
python scraper.py
```
